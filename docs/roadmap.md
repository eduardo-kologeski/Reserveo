# Roadmap

## Curto Prazo

- Criar backend Reserveo com endpoints de clientes, profissionais, servicos e agendamentos.
- Adicionar fixtures e testes unitarios.
- Ajustar AuthCore para emitir roles `CUSTOMER` e `PROFESSIONAL`.

## Medio Prazo

- Criar calendario visual por dia, semana e profissional.
- Adicionar regras de disponibilidade e bloqueios de agenda.
- Adicionar notificacoes por email.
- Adicionar testes e2e.

## Longo Prazo

- Evoluir para multiempresa.
- Criar assinatura SaaS.
- Criar auditoria e relatorios.
- Publicar deploy de demonstracao.
